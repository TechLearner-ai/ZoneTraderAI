# Overview

ZoneTrader AI is a professional AI-powered Equity Decision Support Platform designed for the Indian stock market.

It combines:

- Fundamental Analysis
- Technical Analysis
- Demand & Supply Zones
- Support & Resistance
- Candlestick Analysis
- Chart Pattern Recognition
- Multi-Timeframe Analysis
- Volume Intelligence
- News Sentiment
- Portfolio Analytics
- Backtesting
- Explainable AI

The objective is to help investors identify high-conviction equity opportunities for:

- Swing Trading
- Positional Trading
- Long-Term Investing

This application is **not** an automated trading system.
